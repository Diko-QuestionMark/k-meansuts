1️⃣
Inisialisasi
Langkah pertama K-Means

Pilih K titik pusat (centroid) secara acak dari dataset. Setiap centroid akan menjadi pusat awal dari masing-masing cluster. Pemilihan centroid awal mempengaruhi hasil akhir clustering.

2️⃣
Assignment
Penugasan ke cluster

Setiap data point dihitung jaraknya ke semua centroid menggunakan Euclidean Distance. Data point akan ditugaskan ke cluster dengan centroid terdekat.

3️⃣
Update
Perbarui centroid

Hitung ulang posisi centroid sebagai rata-rata dari semua data point dalam cluster. Ulangi langkah 2 & 3 hingga centroid tidak berubah (konvergen) atau mencapai iterasi maksimum.
